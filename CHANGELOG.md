# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2026-04-30

### Initial Setup
- Initialized project using **Vite**, **React**, and **Tailwind CSS v4**.
- Configured a sidebar-based desktop navigation architecture.
- Integrated **Radix UI** and **shadcn/ui** component patterns for high-fidelity admin interfaces.

### Role & Data Sync
- Removed the redundant 'Coach' role to align with the Mobile Web prototype (v3.0).
- Standardized system roles to exclusively support **'Player'** and **'Organization'**.
- Updated user data structures to include advanced organizational metadata.

### Organization Management
- **Dashboard Item**: Added "Organizations" to the primary navigation with the `Building2` icon.
- **Dedicated Interface**: Created a specialized dashboard for monitoring and managing organization-type accounts.
- **Enhanced Metadata**: Integrated fields for **Official Designation**, **Business Address**, and **Joined Date**.
- **Compliance Labeling**: Updated verification workflows to use "Government Permit or Business License" terminology.

### Activity Tracking
- **Event Drill-Down**: Implemented a "View Events" modal providing a detailed history of hosted games and tournaments.
- **Performance Metrics**: Added a "Hosting Health" dashboard within the events view, tracking completion rates and average participant engagement.

### Bug Fixes
- **Modal Depth & Opacity**: Resolved a critical visual bug where background table elements were bleeding through the modal overlay.
- **Stacking Integrity**: Enforced explicit z-indices (`z-90` for overlay, `z-100` for content) and solid background backgrounds across all management modals.
